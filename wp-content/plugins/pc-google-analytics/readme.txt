Google analytics for WordPress plugin adds the google tracking code on your website.

== Description ==

The Google Analytics plugin easily adds and enables Google Analytics tracking code on your website. 



Features of the plugins

*   Enables Google Analytics for your site.

If you still think that some features are missing then suggest me I will be happy to add those features to the plugin. Your suggestions are always appreciated.


[Author's site](http://lifesoftwares.com/)

== Installation ==

1. Upload `pc-google-analytics` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the Settings > Google Analytics to change setting


== Frequently Asked Questions ==
= Where can I find my Google Analytics tracking code? =

You can create a free "property" on the Google Analytics website. The UA code is the tracking code which is used in this plugin.

For any support issues, please use the official WordPress support forums.
